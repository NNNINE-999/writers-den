# v0.1.0 · 初始发布

**2026-04-30**

## 新增

- 用户注册（邮箱验证码 + QQ 邮箱 SMTP）
- 用户登录 / 登出
- 30 天持久会话（Cookie）
- 文章发布（Markdown 编辑器 + 实时预览）
- 文章详情页（Markdown 渲染）
- 文章编辑
- 文章删除（二次确认）
- 标签系统（逗号分隔输入，自动创建）
- 标签云（侧栏，按文章数排序）
- 标签页（按标签过滤文章）
- 评论发表与删除
- 首页文章流（相对时间显示）
- 邮箱验证码模板（秘书小陈/陈文逸角色）
- 第一个注册用户自动成为管理员

## 技术

- Next.js 16 App Router + Turbopack
- SQLite (Turso / @libsql/client) + Drizzle ORM
- Tailwind CSS v4 暖色调主题
- 部署到腾讯云轻量服务器（Ubuntu 24.04, PM2, Nginx）
