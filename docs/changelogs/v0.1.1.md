# v0.1.1 · 移动端修复

**2026-04-30**

## 修复

- 移除全局 `backdrop-blur`（移动端渲染性能问题）
- 首页文章卡片 `after:absolute` 伪元素 overlay 移除（tap 区域问题）
- 全局添加 `touch-action: manipulation`（消除 300ms 点击延迟）
- `overflow-x: hidden` 防止移动端横向滚动
- Navbar 从 `sticky` 改为 `relative`（移动端定位问题）
- 所有 Navbar Link 添加 `prefetch={false}`（减少移动端 RSC fetch 延迟）
- Cookie `secure` 设为 `false`（HTTP 环境登录修复）
